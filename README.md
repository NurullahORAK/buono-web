# Buono Premium Brand Website

Modern premium pastry brand website built with:

- Next.js (App Router)
- TailwindCSS
- Headless CMS ready architecture (Sanity planned)
- Component-based scalable structure

## Features

- Product listing & detail pages
- Gift box page with WhatsApp CTA
- Production center page (Chef carousel)
- Quality & food safety page (media slider + lightbox)
- Responsive design
- Premium typography system

## Tech Stack

- Next.js 14+
- TailwindCSS
- Embla Carousel
- TypeScript

## Status

Frontend foundation completed.
CMS integration (Sanity) planned.
